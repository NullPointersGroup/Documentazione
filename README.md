# Analisi multimodale per la creazione automatica di ordini
Proponente: Ergon Informatica Srl

## Link
https://www.math.unipd.it/~tullio/IS-1/2025/Progetto/C8.pdf 

## Obiettivo del progetto
Automatizzare la ricezione e l'elaborazione di ordini di acquisto da input multimodali (testo, audio, immagini), trasformandoli in ordini strutturati pronti per l'inserimento in database aziendali.

## Problema affrontato
Le richieste dei clienti arrivano in formati non strutturati e spesso ambigui, rendendo la gestione manuale lenta, costosa e soggetta a errori.

## Soluzione proposta
Una piattaforma intelligente basata su AI, Machine Learning e NLP, in grado di interpretare input multimodali, estrarre informazioni, validarle e generare ordini standardizzati.
