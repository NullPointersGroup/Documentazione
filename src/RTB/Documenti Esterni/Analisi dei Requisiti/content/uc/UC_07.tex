\subsubsection{UC\_07: Visualizza informazioni profilo}
\label{uc_07}

L'UC\_07 ha 3 ulteriori inclusioni indicate in figura.

\begin{figure}[H]
        \centering
        \includegraphics[width=0.8\textwidth]{AnalisiRequisiti/UC_07}
        \caption{Inclusioni UC\_07: UC\_07.1, UC\_07.2, UC\_07.3}
\end{figure}

\begin{itemize}
    \item \textbf{Attore principale}: Cliente
    \item \textbf{Scenario principale}: 
            \begin{enumerate}
                \item L'Utente accede alla sezione info del proprio profilo
                \item L'Utente visualizza il proprio username $\rightarrow$ Vedi \hyperref[uc_07.1]{UC\_07.1}
                \item L'Utente visualizza la propria ragione sociale $\rightarrow$ Vedi \hyperref[uc_07.2]{UC\_07.2}
                \item L'Utente visualizza il proprio indirizzo email $\rightarrow$ Vedi \hyperref[uc_07.3]{UC\_07.3}
            \end{enumerate}
    \item \textbf{Precondizioni}:   
            \begin{itemize}
                \item Il sistema è online
                \item L'Utente è autenticato
            \end{itemize}      
    \item \textbf{Postcondizioni}: 
            \begin{itemize}
                \item Il sistema mostra una pagina HTML con le informazioni Utente
            \end{itemize}
    \item \textbf{Inclusioni}: 
            \begin{itemize}
                \item \hyperref[uc_07.1]{UC\_07.1}
		        \item \hyperref[uc_07.2]{UC\_07.2}
		        \item \hyperref[uc_07.3]{UC\_07.3}
            \end{itemize}
    \item \textbf{User Story}: L'Utente vuole visualizzare i propri dati registrati nel sistema
\end{itemize}
